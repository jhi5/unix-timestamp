J. Munford's Timestamp API

How To Use

Feed the url above a date such as...

http://jmunford-unix-timestamp.herokuapp.com/December%2015,%202015

or

http://jmunford-unix-timestamp.herokuapp.com/1450137600

And you'll get this JSON object back with the Unix and Standard versions of your date!

{"standard":"January 17, 1970","unix":"1450137600"}
